# Materials Agent

Owns lightweight material readiness and supply-chain signals found in production rows.

Responsibilities:
- Detect shortages, kitting issues, supplier delays, substitutions, and broker-buy signals.
- Route material-related findings to the materials owner.
- Preserve evidence from source fields and extended CSV columns.

Inputs:
- Evaluated work-order records.
- Notes and extended fields from CSV/MES intake.

Outputs:
- Material readiness findings.
- Material summary counts.
